🌐 网页翻译弹窗悬停隐藏功能测试

📋 测试说明
本次更新让网页上的翻译弹窗也支持鼠标离开时自动隐藏，与收藏页面的弹窗行为保持一致，提供统一的用户体验。
🧪 测试步骤
1 确保已安装并启用"多多记单词"Chrome扩展
2 在下方测试内容中选中任意英文单词
3 点击出现的📖翻译按钮，观察弹窗显示
4 将鼠标移动到弹窗上，确认弹窗保持显示
5 将鼠标移出弹窗，观察是否在200ms延迟后自动隐藏
6 测试已收藏单词的悬停显示功能
📝 测试内容

This is a comprehensive test for the word translation feature. You can select any appropriate English word to test the translation functionality.

The implementation includes advanced features like pronunciation, definitions, examples, and synonyms. Try selecting words like beautiful, important, or development to see the rich translation data.

The extension also supports intelligent word highlighting and automatic recognition of word variations. For example, words like running, better, or happiness will be properly handled.

Long text translation is also supported. Try selecting this entire sentence to test the text translation feature with the blue 🌐 button.

✅ 预期行为
✓ 选中单词后显示📖翻译按钮
✓ 点击翻译按钮显示详细翻译弹窗
✓ 鼠标在弹窗上时，弹窗保持显示
✓ 鼠标离开弹窗后，200ms延迟后自动隐藏
✓ 已收藏单词悬停时立即显示详情
✓ 发音按钮和收藏功能正常工作
⚠️ 注意事项
! 弹窗位置会根据屏幕边界自动调整
! 首次翻译可能需要加载时间，会显示加载状态
! 鼠标快速移动可能导致弹窗闪烁，这是正常现象
! 长文本翻译使用🌐按钮，单词翻译使用📖按钮
🔧 技术改进对比
v1.7.5 之前：
  • 网页弹窗需要点击外部才能关闭
  • 收藏页面弹窗支持悬停隐藏
  • 两种弹窗行为不一致
v1.7.6 现在：
  • 网页弹窗支持鼠标离开时自动隐藏
  • 收藏页面弹窗保持悬停隐藏功能
  • 两种弹窗具有统一的交互体验
  • 使用data-hover属性精确控制状态
